# SalesSavvy

![SalesSavvy Banner](https://via.placeholder.com/1200x400/1e3a8a/ffffff?text=SalesSavvy+-+Sales+Management+System)

## 📋 Project Overview

**SalesSavvy** is a modern **full-stack web application** designed to help businesses efficiently manage their sales, customers, products, and inventory.

Built with a robust Java Spring Boot backend and a clean, responsive frontend using HTML, CSS, and JavaScript, this project provides a complete solution for tracking sales orders, maintaining customer records, and managing product inventory.

### ✨ Key Features

- User Registration and Login system
- Interactive Dashboard with sales overview
- Customer Management (CRUD operations)
- Product & Inventory Management
- Sales Order Processing
- Responsive and user-friendly interface
- RESTful APIs for seamless frontend-backend communication

## 🛠️ Tech Stack

### Backend
- **Java 17**
- **Spring Boot Framework**
- **Spring Data JPA**
- **Hibernate** (ORM)
- **MySQL** Database
- **Maven** (Build Tool)
- REST APIs with proper validation and exception handling

### Frontend
- **HTML5**
- **CSS3** (with modern styling)
- **JavaScript** (Vanilla + Fetch API / AJAX)
- Fully Responsive Design
