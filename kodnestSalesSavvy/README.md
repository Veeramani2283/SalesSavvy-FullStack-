# Sales Savvy – Modular E-Commerce Backend

### Tech Stack
**Java, Spring Boot, REST APIs, JWT Authentication, MySQL, Razorpay Integration, Docker**

### Key Engineering Highlights
* **Service Architecture:** Developed six independent modules: Authentication, User Management, Product, Cart, Order, and Payment.
* **Security & RBAC:** Implemented secure authentication and role-based access control with JWT for Admins and Customers.
* **API Design:** Built RESTful endpoints for registration, product catalog, cart operations, and order lifecycle management.
* **Payment Integration:** Integrated third-party gateways (Razorpay/Stripe) to handle secure transactions and confirmations.
* **Data Integrity:** Applied best practices in password hashing (BCrypt), data validation, and session security.
* **Database Optimization:** Designed a relational schema ensuring smooth transitions from **Cart → Order → Payment**.
